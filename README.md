# Frame

Коллекционная карточная игра с фильмами и сериалами. Работает прямо в браузере, без установки — карточки, паки, магазин и всё остальное хранится локально у тебя в браузере.

Открыть: https://zimali.github.io/Frame/

## Что это

Открываешь паки, получаешь карточки с фильмами и сериалами (обложки и описания подтягиваются с TMDB), собираешь коллекцию, продаёшь лишнее, покупаешь новое в магазине. За активность даётся опыт и уровень, есть ежедневные задания и значки за прогресс.

## Разделы

**Открыть** — тут открываются паки. При открытии карточка сначала быстро прокручивается среди других, потом останавливается на результате. Можно тапнуть карту, чтобы перевернуть её и посмотреть описание.

**Коллекция** — все твои карточки. Есть поиск, сортировка (по новизне, по алфавиту), фильтр по редкости и по избранному. У каждой карточки свой номер по порядку получения. При наведении на карточку показывается подсказка с названием, редкостью и номером. Долгое нажатие на кнопку "Коллекция" открывает список коллекций — можно создать свою (платно, монетами), переименовать, удалить или перекинуть туда карточки.

**Магазин** — лоты с карточками за монеты, обновляются по таймеру.

**Прогресс** — уровень, опыт, ежедневные задания, полоса стриков (сколько дней подряд заходишь), значки за разные достижения (сколько паков открыто, карточек продано, куплено и т.д.) и отдельная плашка с прогрессом до алмазной карточки — она выдаётся, когда выполнены все три ежедневных задания.

**Настройки** — язык интерфейса (десять штук), звук и музыка, оттенок фона, иконка профиля, включение/выключение подсказок при наведении на карточки. Тут же сброс прогресса и повтор обучения.

## Карточки

Редкости: обычная, золотая, радужная, уникальная, алмазная — по возрастанию редкости и ценности. Карточку можно добавить в избранное, продать за монеты или перенести в свою коллекцию. Есть режим множественного выбора — зажимаешь кнопку рядом с поиском и можно сразу выделить несколько карточек и продать их или перенести все разом.

При просмотре карточки крупным планом можно покрутить её мышкой/пальцем, включить автоматическое покачивание или крутить её через кнопку.

## Технически

Обычный HTML/CSS/JS без сборки и фреймворков. Картинки и описания фильмов — через API TMDB. Весь прогресс хранится в localStorage браузера, поэтому если чистить кэш или заходить с другого устройства — прогресс не сохранится, он привязан к конкретному браузеру.
